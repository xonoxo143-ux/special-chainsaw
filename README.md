# Android Builder Lab

System-first, phone-friendly Android Builder / Editor / Installer Lab.

## Core idea
Build an Android app that helps manage Android projects from a phone by combining:
- Android system pickers and installer flows
- workspace/staging for reliable mobile handling
- a basic in-app file browser/editor
- Termux as the required execution backend for heavy build tasks

## Current status
- Planning locked
- GitHub repo initialized for project tracking
- Build 1 is the active starting milestone

## Delivery model
- `FULL` builds become new baselines
- `PATCH` builds are for isolated fixes only
- `WIPE / RESET` builds are used when the structure goes bad

## First milestone
`Build 1 — Foundation Shell`

Goal:
- open something
- detect what it is
- stage it if needed
- show the recommended path cleanly
